## TF2Autobot GUI Panel v3.5.9

### Pricedb pricelist array compatibility

- **Shared** `getPricelistEntries()` normalizes pricelist IPC from [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) (array format) for Unlisted Stock filtering and the pricelist API route
- Pricelist type accepts both **array** and keyed-object formats from bot IPC
- Release restore script can replace `cursor[bot]` GitHub releases when run as **uwu6967**

### Update

```bash
git pull
npm install
npm run build
pm2 restart tf2autobot-gui
```

Also ensure your bot is on the latest [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) release (v1.0.4+).
