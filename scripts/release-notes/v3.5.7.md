## TF2Autobot GUI Panel v3.5.7

### Unlisted Stock pricelist filtering fix

- **Fixed** Unlisted Stock showing items that are already on your pricelist when using [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb)
- Pricelist IPC from pricedb sends an **array** of entries — the panel now handles both array and keyed-object formats
- Actively listed items (enabled with buy/sell intent) are correctly excluded from the unlisted view

### Update

```bash
git pull
npm install
npm run build
pm2 restart tf2autobot-gui
```

Also ensure your bot is on the latest [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) release.
