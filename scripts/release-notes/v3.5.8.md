## TF2Autobot GUI Panel v3.5.8

### Unlisted Stock listing feedback

- **Fixed** List for sale and bulk list actions on the Unlisted Stock page
- Bot error strings from [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) IPC are surfaced in the UI instead of failing silently
- Already-priced items are upserted (add falls back to update) so re-listing from Unlisted Stock works
- Row animations show listing progress, success removal, and failure states

### Update

```bash
git pull
npm install
npm run build
pm2 restart tf2autobot-gui
```

Also ensure your bot is on the latest [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) release (v1.0.4+).
