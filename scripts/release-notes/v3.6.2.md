## TF2Autobot GUI Panel v3.6.2

### Asset-id IPC, panel self-update fixes, and version checks

- **Asset ids over IPC** — single-asset unlisted items are listed with their backpack asset id so tf2autobot-pricedb uses the correct `priceKey`
- **Asset-keyed pricelist lookup** — `findPricelistEntryBySkuOrId` matches pricelist entries by asset id or SKU
- **Panel self-update** — update script uses PM2 process name `panel` by default; stale update jobs are marked failed after 30 minutes
- **Version verification** — `npm run verify` checks version sync, builds, and pricedb compatibility smoke tests
- **Pricedb compatible** — tested with [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) v1.0.4+

### Update

```bash
git pull
npm install
npm run build
pm2 restart panel
```

Also ensure your bot is on the latest [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) release (v1.0.4+).
