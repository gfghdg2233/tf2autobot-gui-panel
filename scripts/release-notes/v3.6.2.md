## TF2Autobot GUI Panel v3.6.2

### Autoprice sell-only / buy-only and panel self-update fixes

- **Partial autoprice toggles** — price modal adds sell-only and buy-only autoprice switches for mixed manual/live pricing
- **Pricedb compatible IPC** — panel translates partial autoprice to `autoprice` + `isPartialPriced` before sending items to [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) (v1.0.5+)
- **Live price fill** — missing autopriced sides are prefilled from pricedb.io before IPC save
- **PM2 self-update** — panel auto-update uses the correct PM2 process name and recovers stale update jobs after 30 minutes
- **Version verification** — `npm run verify` checks version sync, build, and pricedb smoke tests

### Update

```bash
git pull
npm install
npm run build
pm2 restart tf2autobot-gui
```

Also ensure your bot is on the latest [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) release (v1.0.5+).
