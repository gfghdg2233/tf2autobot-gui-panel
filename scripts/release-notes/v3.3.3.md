## TF2Autobot GUI Panel v3.3.3

Web panel for managing your TF2Autobot trading bot — pricelist, trades, settings, profit tracking, and live backpack.tf reference prices.

### Quick start

```bash
git clone https://github.com/uwu6967/tf2autobot-gui-panel.git
cd tf2autobot-gui-panel
npm install
cp template.env .env
npm run build
npm start
```
