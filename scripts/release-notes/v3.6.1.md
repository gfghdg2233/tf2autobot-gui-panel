## TF2Autobot GUI Panel v3.6.1

### Unlisted list fixes for asset-keyed and manual-priced items

- **Asset ids over IPC** — unique items are listed with their backpack asset id so tf2autobot-pricedb can price them correctly
- **Pricelist upsert** — existing pricelist entries are merged on add/update instead of failing with "already priced"
- **Real bot errors** — error strings from tf2autobot-pricedb `addItem` / `updateItem` are shown in the UI
- **Smarter listing queue** — items are only queued when autoprice is unavailable, not for unrelated failures
- **Pricedb compatible** — tested with [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) v1.0.4+

### Update

```bash
git pull
npm install
npm run build
pm2 restart tf2autobot-gui
```

Also ensure your bot is on the latest [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) release (v1.0.4+).
