## TF2Autobot GUI Panel v3.6.0

### Listing queue and manual pricelist updates

- **Listing Queue** tab on the Pricelist page shows failed unlisted items waiting to be listed manually
- **Autoprice fallback** — when tf2autobot-pricedb cannot autoprice an item, the panel retries with manual pricing
- **Per-bot queue** persists failed unlisted items under `data/listing-queue/` and clears entries after a successful list
- **Pricedb compatible** — works with [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) v1.0.4+ array pricelist IPC and string error responses from `addItem` / `updateItem`

### Update

```bash
git pull
npm install
npm run build
pm2 restart tf2autobot-gui
```

Also ensure your bot is on the latest [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) release (v1.0.4+).
