## TF2Autobot GUI Panel v3.4.1

### Discord update webhook restored

The panel can again post **Mann Co. styled Discord embeds** when a new GUI version is released.

**Setup**

1. Create a webhook in Discord (Server Settings → Integrations → Webhooks)
2. Add to your `.env`:
   ```env
   DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/...
   ```
3. Start the panel — it posts **once per new version** automatically

**Also includes**

- `DISCORD_WEBHOOK_ENABLED` and `DISCORD_WEBHOOK_ALWAYS` options
- Links to [tf2autobot-pricedb](https://github.com/uwu6967/tf2autobot-pricedb) in the embed
